## Prerequisites

Before starting, ensure you have the following prerequisites installed on your system.

<Tabs>
  <Tab title="macOS" icon="monitor">
    Install dependencies using Homebrew:
    
    ```bash
    brew install node npm
    brew install go
    ```
    
    <Callout kind="info">Node.js version 16+ and Go 1.18+ are required for Oasis development.</Callout>
  </Tab>
  <Tab title="Linux" icon="terminal">
    Install dependencies using your package manager:
    
    ```bash
    sudo apt update
    sudo apt install nodejs npm golang-go
    ```
  </Tab>
  <Tab title="Windows" icon="monitor">
    Use Chocolatey for package management:
    
    ```powershell
    choco install nodejs
    choco install golang
    ```
  </Tab>
</Tabs>

## Installing Oasis SDK

<Steps>
  <Step title="Initialize Project" icon="folder">
    Create a new project directory and initialize npm:
    
    ```bash
    mkdir oasis-app
    cd oasis-app
    npm init -y
    ```
  </Step>
  <Step title="Install SDK" icon="download">
    Install the Oasis Protocol SDK:
    
    ```bash
    npm install @oasisprotocol/client
    npm install @oasisprotocol/contract
    ```
  </Step>
  <Step title="Configure Network" icon="settings">
    Set up network configuration:
    
    ```javascript
    const oasis = new OasisClient({
      gateway: 'https://sapphire.oasis.io',
      network: 'sapphire'
    });
    ```
  </Step>
  <Step title="Test Connection" icon="check">
    Verify your connection to the Oasis network:
    
    ```javascript
    const blockNumber = await oasis.getBlockNumber();
    console.log('Current block:', blockNumber);
    ```
  </Step>
</Steps>

## Deploying Your First Contract

<Expandable title="Simple Confidential Contract" default-open="true">
  Let's deploy a simple confidential smart contract that stores encrypted data.
  
  <CodeGroup tabs="Solidity,JavaScript">
    ```solidity
    // SPDX-License-Identifier: MIT
    pragma solidity ^0.8.0;
    
    contract ConfidentialStorage {
      mapping(address => bytes32) private data;
      
      function storeData(bytes32 encryptedData) external {
        data[msg.sender] = encryptedData;
      }
      
      function getData() external view returns (bytes32) {
        return data[msg.sender];
      }
    }
    ```
    ```javascript
    // Deploy contract
    const contract = await oasis.deployContract({
      bytecode: compiledBytecode,
      abi: contractAbi,
      confidential: true
    });
    
    console.log('Contract deployed at:', contract.address);
    
    // Interact with contract
    const encryptedData = oasis.encrypt('secret data');
    await contract.storeData(encryptedData);
    ```
  </CodeGroup>
</Expandable>
